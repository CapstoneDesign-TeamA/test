📌 주요 기능 정리 (통합 버전)
1. 회원 & 권한 관리

-소셜 로그인: 구글, 네이버, 카카오 (개발 난이도 ↓)
-GroupMembers 테이블 기반 역할 부여: Leader / Member (MVP)
-확장 시 총무, 관리자 등 추가 가능
-권한 차등:
	- 회비 수정 가능 여부
	- 모임원 강퇴 가능 여부
	- 일정/사진첩 관리 권한 구분

2. 모임 관리 & 회비

- 여러 모임 생성 가능: 동창, 동호회, 취미 모임 등
- 회비 관리: 납부일, 알림, 미납자 확인
- 결제 API 연동:
	- 카카오페이, 토스페이먼츠 API로 회비 송금
	- 일정 후 “1/n 계산” 기능 → 자동 송금 요청 → 총무 계좌에 입금
	- 활동 계획 공유: 시간표, 비용 분담, 장소 안내


3. 일정 & 위치 공유

- 기본 일정 공유: 그룹 캘린더에 등록
- 외부 캘린더 연동: Google Calendar, 네이버 캘린더
- 위치 API 활용: 약속 장소 확정 → 지도 공유 → 모임원에게 푸시 알림

4. 사진첩 & 추억 관리

- 사진첩: 모임별로 사진/영상 업로드 & 날짜별 정렬
- AI 태그/분류: 바다, 산, 건물 등 자동 태그 → 관련 장소/활동 추천 가능
- 타임캡슐: 특정 날짜까지 잠금 → 날짜 도래 시 그룹원 전체 열람 가능

5. 추천 & 네트워킹

- AI 추천: 사진 기반 장소/활동 추천
- 주변 할인 지점 API 연동
- 친구 관리: 친구 추가/삭제, 취향 기반 친구 추천
- 공개 번개 모임: 모집 게시글 작성 → 성비/참여 조건 설정 가능
