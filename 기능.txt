사진첩 : 카테고리로 따로 빼서 업로드한 사진이나 동영상 따로 모아두고 날짜별로 정렬하는 기능(업로드 날짜도 출력)

카카오페이, 토스페이멘츠 api 기능(입금,송금 등) 이용해서 회비 납부, or 일정 후에 1/n 계산 요청 후 총무에게 송금(가능할 듯)
-> 소셜로그인 사용하면 개발 더 쉬워질지도

지도 공유 : 약속 장소 결정 후 위치api 이용해서 지도 공유

타임캡슐 : 추억이 될만한 사진 내지 정보를 특정 날짜까지 잠금, 특정 날짜가 되면 그룹원 모두 열람 가능



공유 캘린더 기반 UI 아이디어

달력 중심 인터페이스
- 메인 화면 자체가 달력 형태
- 날짜별로 그룹 활동과 개인 할 일을 한눈에 확인 가능

할 일 관리
- 미래 일정: 개인/그룹 할 일을 달력에 기록
- 일정에 다다르면 푸시 알림, 완료 체크 가능
- 지난 기록 표시

지난 날짜 칸에는 그날 업로드한 사진 미리보기 표시
- 날짜 클릭 시 상세 게시물 화면으로 이동

상세 게시물 화면 구성 (날짜별 그룹 기록)
- 활동 사진첩 (그룹에서 올린 사진/영상)
- 회비 내역 (그날의 지출/정산 내역)
-  댓글 기능 (그룹 멤버 간 소통 공간)
